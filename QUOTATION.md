## Project Summary

JP Style Lounge Studio is a mobile-first barber booking app designed for a solo barber operation in Accra, with a future-ready multi-tenant foundation. The Phase 1 MVP focuses on helping clients book appointments, pay deposits, receive reminders, and leave reviews, while giving the barber tools to manage schedule and revenue.

---

## Scope Included

The quoted amount includes the following Phase 1 MVP deliverables:

1. Flutter mobile app setup for Android and iOS
2. Supabase backend setup and database schema
3. Authentication flows
   - Phone OTP
   - Email/password
   - Guest booking
4. Barber profile and services display
5. Availability and calendar booking system
6. 5-step booking flow
7. Paystack payment integration
8. Push notification setup
9. SMS fallback integration hooks
10. Cancellation and no-show logic
11. Reviews and ratings system
12. Barber dashboard for bookings and revenue overview
13. Basic deployment preparation and launch support

---

## Scope Excluded

The following items are not included in this Phase 1 MVP quotation unless separately agreed:
1. Multi-barber marketplace features
2. Loyalty points and referral systems
3. Full web dashboard build
4. Advanced analytics and reporting beyond MVP level
5. Ongoing maintenance after delivery
6. App Store and Play Store publishing fees
7. Paid design/branding work beyond standard product UI
8. Third-party paid service charges
