# App Store Privacy Nutrition Label Matrix

Status: Draft baseline for JP Style Lounge Studio
Last updated: 2026-04-30

## Data Types and Usage Intent

| Data Type | Linked to User | Used for Tracking | Purpose |
|---|---|---|---|
| Contact Info (name, phone, email) | Yes | No | Account and booking operations |
| User Content (uploaded photos/reviews) | Yes | No | Booking/review features |
| Identifiers (user id, push token) | Yes | No | Auth/session and notifications |
| Location (when enabled) | Yes | No | Maps and directions |
| Purchases/Payment state | Yes | No | Booking payment confirmation |

## Tracking Declaration Baseline

- Tracking: No
- Third-party advertising tracking: No
- Cross-app tracking identifiers: No
